## Systèmes POS

Connectez votre système de point de vente pour suivre automatiquement les ventes de salsa et l'utilisation de l'inventaire.

### Intégration Square


1. **Activer l'intégration Square**
   Naviguez vers **Intégrations > Systèmes POS** dans votre tableau de bord.

2. **Connecter votre compte Square**
   Cliquez sur "Connecter Square" et autorisez l'intégration avec vos identifiants Square.

3. **Mapper les articles du menu**
   Mappez vos articles de menu de salsa aux articles d'inventaire correspondants dans Salsa as a Service.

4. **Tester la connexion**
   Traitez une transaction de test pour vérifier que l'intégration fonctionne correctement.



**Fonctionnalités :**
- Suivi automatique des ventes
- Mises à jour d'inventaire en temps réel
- Analytique des commandes clients
- Rapports de revenus

> **Note:**
**Exigences de l'API Square** : Assurez-vous que votre compte Square a l'accès API activé et les permissions nécessaires pour la gestion d'inventaire.


### Intégration Toast

L'intégration Toast POS offre des fonctionnalités avancées pour les restaurants à service complet :

**Processus de configuration :**

1. **Configuration API** : Configurez vos identifiants API Toast

3. **Mappage d'inventaire** : Mappez l'inventaire Toast aux articles Salsa as a Service

## Intégrations personnalisées

### Configuration de webhooks

Configurez des webhooks pour la synchronisation des données en temps réel :

```javascript
// Exemple de gestionnaire de webhook
app.post('/webhooks/salsa', (req, res) => {
  const { event, data } = req.body;

  switch (event) {
    case 'production.batch_completed':
      updateKitchenDisplay(data);
      break;
    case 'inventory.low_stock':
      sendAlertToManager(data);
      break;
    case 'quality.alert':
      notifyQualityTeam(data);
      break;
  }

  res.status(200).send('OK');
});
```

### Intégration API

Créez des intégrations personnalisées en utilisant notre API REST :

**Authentification :**
```bash
curl -H "Authorization: Bearer YOUR_API_KEY" \
     https://api.salsaasaservice.com/v1/recipes
```

**Limitation de débit :**
- Plan gratuit : 1 000 requêtes/heure
- Professionnel : 10 000 requêtes/heure
- Entreprise : 100 000 requêtes/heure

## Dépannage

### Problèmes d'intégration courants

> **Warning:**
**Limites de débit API** : Si vous atteignez les limites de débit, envisagez de mettre en œuvre un recul exponentiel ou de mettre à niveau votre plan.


**Erreurs d'authentification :**
- Vérifiez que la clé API est correcte
- Vérifiez les permissions de la clé
- Assurez-vous que la clé n'a pas expiré

**Problèmes de synchronisation des données :**
- Vérifiez la connectivité réseau
- Vérifiez les points de terminaison webhook
- Examinez les journaux de synchronisation
- Testez avec l'environnement sandbox

**Problèmes de performance :**
- Surveillez les temps de réponse de l'API
- Vérifiez la limitation de débit
- Optimisez la fréquence des requêtes
- Envisagez des stratégies de mise en cache
